🔗 Приложение 5a: SPINE → Anchors Mapping

📋 Назначение: Эта секция описывает связь между SPINE элементами и психологическими якорями. Используйте её как справочник при создании карточки для обеспечения когерентности персонажа.

📊 Определения якорей

ЯкорьОпределениеПример
Growth Anchor
(Якорь роста)		Положительный результат/осознание, к которому персонажу нужно прийтиСамопринятие без перфекционизма
Avoidance Anchor
(Якорь избегания)		То, от чего персонаж бежит любой ценойУязвимость и близость
Origin Memory
(Первоначальная память)		Формирующее событие, которое заложило базовые убежденияВ 7 лет родители развелись; персонаж винил себя
Core Belief (CBT)
(Базовое убеждение)		Когнитивное искажение, формирующее интерпретациюОшибки означают, что я недостоин любви
Attachment Style
(Стиль привязанности)		Паттерн отношений из раннего уходаТревожно-избегающий

📝 Пример: Персонаж «Елена Васкес» (Тип 1w2)

SPINE:

FLAW: Должна быть права, чтобы быть в безопасности
LIE: Если я ошибусь, меня бросят
GHOST: В 9 лет отец ушёл после ссоры, где она заняла не ту сторону
PRICE: Самопроизвольность, радость, близкие отношения
WANT: Доказать свою незаменимость через компетентность
NEED: Принять, что ошибки не означают отсутствие любви

OCEAN: O:72 C:85 E:28 A:45 N:62

Anchors:

Growth Anchor: Отпустить потребность быть правой
Avoidance Anchor: Уязвимость, ошибаться
Origin Memory: Ночь, когда ушёл отец
Core Belief: Ошибки = Одиночество
Attachment Style: Избегающий

Mapping:

  • FLAW → Enneagram Fixation (Тип 1: Перфекционизм) + OCEAN Low E (Избегание внимания)
  • LIE → Core Belief (Ошибки = Одиночество) + Enneagram Passion (Гнев)
  • GHOST → Origin Memory (Уход отца) + Attachment Style (Избегающий)
  • PRICE → Avoidance Anchor (Уязвимость) + Enneagram Desire (Быть хорошей)
  • WANT → Enneagram Desire (Быть хорошей) + OCEAN High C (Контроль через компетентность)
  • NEED → Growth Anchor (Принятие несовершенства) + Enneagram Virtue (Спокойствие)

✅ Чеклист валидации SPINE-Anchors

  • ☐ Каждый SPINE элемент связан хотя бы с одним якорем
  • ☐ GHOST объясняет происхождение LIE
  • ☐ LIE поддерживает FLAW
  • ☐ FLAW блокирует NEED
  • ☐ WANT противоположно или конфликтует с NEED
  • ☐ Avoidance Anchor связан с FLAW или GHOST
  • ☐ Growth Anchor описывает результат принятия NEED
  • ☐ OCEAN экстремальные полюса поддерживают FLAW или WANT
  • ☐ Enneagram тип соответствует паттерну SPINE
