⚠️ Самые частые ошибки новичков:
  1. presence_penalty = 0.7 → сбрось на 0.0 (для 12B это критично)
  2. Голос в Description → удали, перенеси в Examples
  3. GHOST в SP → перенеси в Description
  4. temperature > 0.85 → снизь до 0.6–0.8 (для 12B)
  5. Нет антигодмодинга → добавь обязательный блок
💡 Правило одного изменения:

Меняй только один параметр за раз. Иначе не узнаешь, что помогло.
