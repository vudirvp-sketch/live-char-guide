📋 Каноническая терминология

Канонический терминАббревиатураОпределениеЗапрещённые варианты
Author's Notes AN Динамический слой контекста для длинных сессий. Обновляется каждые 3–5 сообщений. «Авторские заметки», «Авторка», «Авторка Заметки»
System Prompt SP Системные инструкции для модели: ограничения, запреты, формат. «Системный промпт», «Системка», «Системник»
Lorebook LB База знаний с триггерами для активации контента по ключевым словам. «LB», «World Info», «WI»
Character Card — Полный пакет поведенческого движка персонажа. «Карточка», «Персонаж»
Description — Описание персонажа (Character Description). Содержит Spine, якоря, фактологию. «Описание», «Character Description»
Examples — Примерные сообщения. Задают голос персонажа через демонстрацию. «Примерные сообщения», «Примеры», «Dialogue Examples»
Greeting — Первое сообщение персонажа в чате. Задаёт начальное состояние и голос. «Первое сообщение», «First Message», «Приветствие»

📍 Расширенные определения по слоям

OCEAN (Big Five) — Layer 1

Модель личности из пяти измерений: Openness (открытость), Conscientiousness (добросовестность), Extraversion (экстраверсия), Agreeableness (доброжелательность), Neuroticism (нейротизм).

Layer 1: Определите 1-2 экстремальных полюса (значения <30 или >70) для поведенческой последовательности.

Полная конфигурация с валидатором: Layer 2+ → Раздел 4.3

Enneagram (базовая конфигурация) — Layer 2 Layer 2+

Layer 2 включает:

  • Идентификация доминантного типа (1-9)
  • Влияние крыла (adjacent types)
  • Базовая мотивация + базовый страх

Layer 3 добавляет: Пути интеграции/дезинтеграции, инстинктивные варианты, полная матрица взаимодействия 9 типов.

Полная документация: Раздел 4.2

Диагностический сценарий (пример) — Layer 2+ Layer 2+

Сценарий: Сообщение пользователя противоречит установленной черте характера (трейту) персонажа.

Обнаружение: LB триггер проверяет флаг консистенции черты характера.

Ответ: Персонаж признаёт расхождение в-характере, сохраняя установленные черты.

Полный диагностический набор: Layer 3 → Раздел 6

⚙️ Технические термины

ТерминОпределениеСвязанный раздел
Price Немедленная физическая или вербальная манифестация последствия. Обязательна для FLAW/GHOST якорей. Раздел 3.1
RepPen Repetition Penalty — штраф за повторы. Рекомендуется 1.05. Выше 1.10 — крайняя мера. Раздел 5.1
CoT Chain of Thought — цепочка рассуждений. Для 12B — базовая строка в SP. Для 32B+/API — не включать. Раздел 5.4
годмодинг Модель пишет за пользователя (действия, мысли, чувства {{user}}). Раздел 5.0.1
OOC Out of Character — выход из образа. Раздел 6
4K-Fallback Протокол адаптации карточки для моделей с ограничением контекста ~4096 токенов. Раздел 5.12
Structured Inject Механизм динамического обновления мотивации через AN или LB с XML-тегами. Раздел 5.3.1
Триггер Ситуация или стимул, активирующий поведенческий якорь. Раздел 3.1
Действие Наблюдаемое поведение персонажа в ответ на триггер. Раздел 3.1
