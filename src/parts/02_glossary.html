📍 Расширенные определения по слоям

OCEAN (Big Five) — Layer 1

Модель личности из пяти измерений: Openness (открытость), Conscientiousness (добросовестность), Extraversion (экстраверсия), Agreeableness (доброжелательность), Neuroticism (нейротизм).

Layer 1: Определите 1-2 экстремальных полюса (значения <30 или >70) для поведенческой последовательности.

Полная конфигурация с валидатором: Layer 2+ → Раздел 4.3

Enneagram (базовая конфигурация) — Layer 2 Layer 2+

Layer 2 включает:

  • Идентификация доминантного типа (1-9)
  • Влияние крыла (adjacent types)
  • Базовая мотивация + базовый страх

Layer 3 добавляет: Пути интеграции/дезинтеграции, инстинктивные варианты, полная матрица взаимодействия 9 типов.

Полная документация: Раздел 4.2

Диагностический сценарий (пример) — Layer 2+ Layer 2+

Сценарий: Сообщение пользователя противоречит установленной черте характера (трейту) персонажа.

Обнаружение: LB триггер проверяет флаг консистенции черты характера.

Ответ: Персонаж признаёт расхождение в-характере, сохраняя установленные черты.

Полный диагностический набор: Layer 3 → Раздел 6
