🚫 Анти-паттерны

Core — критически важно избегать этих ошибок

📍 Position: Read this section BEFORE diagnostics. Knowing anti-patterns helps identify issues faster.
📋 Scope этого раздела: Универсальные паттерны проектирования персонажей (Mary Sue, Token Bloat, вложенные якоря). Для карточко-специфичных симптомов и диагнозов → см. Раздел 6: Диагностика.
⚠️ Зачем этот раздел: Гайд показывает правильные подходы, но не менее важно избегать распространённых ошибок. Каждый анти-паттерн здесь — из реального опыта отладки карточек.

❌ Анти-паттерн 1: Перегрузка OCEAN

— Использование всех 5 полюсов вместо 1–2 экстремумов → «усреднённый» персонаж

Проблема: Использование всех 5 полюсов OCEAN вместо 1–2 экстремумов.

❌ Плохо✅ Хорошо
 
O:45 C:62 E:38 A:55 N:48
O:72 C:85 E:28 A:45 N:62

Все 5 значений. Модель не знает, что подчеркнуть.

 
OCEAN: O:72 N:62
(2 экстремума: Высокая открытость, Высокий нейротицизм)

1–2 экстремума создают запоминающийся профиль.

Симптом: Персонаж «усреднённый», без характерных черт, голос «плывёт».

Решение: Оставь только 1–2 полюса с крайними значениями (<30 или >70). Удали остальные. [→ Правило §1]

❌ Анти-паттерн 2: GHOST в System Prompt по умолчанию

— Размещение GHOST в SP → утечка инструкций, нестабильное поведение

Проблема: Размещение GHOST в System Prompt в стандартной конфигурации.

❌ Плохо✅ Хорошо
 
[System Prompt]
...
GHOST: В детстве был предан близким другом...

GHOST в SP — утечка инструкций.

 
[Description]
<motivation>
GHOST: В детстве был предан близким другом...
</motivation>

[Lorebook]
keyword: "друг|предательств"
content: [GHOST-триггер]

GHOST в Spine, триггеры в LB/AN.

Симптом: Модель «спамит» GHOST-реакциями, поведение нестабильное, путает приоритеты.

Решение: Следуй карте слоёв GHOST. SP = только стиль и запреты. [→ Условное размещение GHOST]

❌ Анти-паттерн 3: Вложенные якоря

— Множественные якоря, триггирующие друг друга → непредсказуемость

Проблема: Множественные якоря, триггирующие друг друга.

❌ Плохо✅ Хорошо
 
- Когда кто-то лжёт → не доверяет → триггер якоря X
- Якорь X: Когда не доверяет → избегает → триггер якоря Y
- Якорь Y: Когда избегает → ...

Каскад якорей = непредсказуемость.

 
- Когда кто-то лжёт → сжимает челюсть, молчит → Цена: дрожат руки

Один триггер → одно действие → одна цена.

Симптом: Непредсказуемое поведение, модель «зацикливается» на триггерах.

Решение: Каждый якорь — атомарная единица. Нет цепочек триггеров. Цена = конечная точка.

❌ Анти-паттерн 4: Token Bloat

— Description более 1000 токенов с избыточными деталями → потеря фокуса

Проблема: Description более 1000 токенов с избыточными деталями.

❌ Плохо✅ Хорошо
 
У неё длинные каштановые волосы с лёгкими локонами,
которые падают на плечи, когда она наклоняется.
Её глаза — глубокий карий с золотистыми крапинками...
[+500 токенов описания внешности]

Пассивное описание без действия.

 
<behavior>
- При волнении → накручивает локон на палец
- Когда смущена → отводит взгляд, каштановые волосы падают на лицо
</behavior>

Детали только если влияют на поведение.

Симптом: Модель теряет фокус, голос «плывёт», приоритет инструкций снижается.

Решение: Удаляй всё, что не генерирует наблюдаемое действие, реакцию или цену.

❌ Анти-паттерн 5: Отложенная цена

— Price не манифестируется немедленно → «слишком идеальный» персонаж

Проблема: Price не манифестируется немедленно в сцене.

❌ Плохо✅ Хорошо
 
- Когда ему лгут → чувствует, но не говорит
  → Цена: потом будет жалеть об этом

«Потом» — не немедленная манифестация.

 
- Когда ему лгут → чувствует, но не говорит
  → Цена: сжимает челюсть, голос становится тише

Физическая манифестация в текущей сцене.

Симптом: Персонаж «слишком идеальный», FLAW не имеет последствий.

Решение: Цена = немедленная физическая или вербальная манифестация. Не «позже», не «навсегда», а сейчас. [→ Правило §2]

❌ Анти-паттерн 6: Голос в Description (для 12B)

— Стилистические директивы в Description для ≤14B → конфликт с Examples

Проблема: Стилистические директивы в Description для моделей ≤14B.

❌ Плохо (12B)✅ Хорошо (12B)
 
<speech_style>
Говорит короткими фразами, использует сленг.
Часто перебивает собеседника.
</speech_style>

Описание голоса в Description.

 
[Description — только логика поведения]

[Examples]
<START>
{{char}}: «Чё, серьёзно?» — он дёрнул плечом. «Ну, давай, рассказывай. Время не ждёт.»

Голос демонстрируется в Examples.

Симптом: Для 12B: конфликт с Examples или игнорирование директив.

Решение: Для ≤14B: удали все стилистические директивы из Description. Голос — только в Examples. [→ Правило §3]

❌ Анти-паттерн 7: Годмодинг в LB content

— Действия/мысли {{user}} в поле content LBа → нарушение user agency

Проблема: Действия/мысли {{user}} в поле content LBа.

❌ Плохо✅ Хорошо
 
content: "{{user}} чувствует влечение к {{char}}
и хочет приблизиться."

Навязывание состояния пользователю.

 
content: "{{char}} замечает интерес {{user}}.
Реакция: наклоняется ближе, улыбается."

Только наблюдаемое поведение {{char}}.

Симптом: Нарушение user agency, модель контролирует {{user}}.

Решение: Никогда не вставляй действия/мысли {{user}} в LB. Это антигодмодинг. [→ Антигодмодинг]

❌ Анти-паттерн 8: RepPen > 1.10

— Высокий RepPen стирает легитимные паттерны → потеря характерных фраз

Проблема: Высокий RepPen стирает легитимные паттерны.

❌ Плохо✅ Хорошо
 
repetition_penalty: 1.15

Штрафует все повторения, включая тики.

 
repetition_penalty: 1.05
# Повышай до 1.08–1.10 только при явных повторах

Стартовое значение. Аккуратное повышение.

Симптом: Персонаж избегает собственных паттернов речи, теряет характерные фразы.

Решение: Старт с 1.05. Повышение только при явных повторах. Никогда выше 1.10 без крайней необходимости. [→ Раздел 5.1]

📋 Чеклист проверки на анти-паттерны

  • ☐ OCEAN содержит только 1–2 экстремальных полюса
  • ☐ GHOST не в System Prompt (кроме 4K-Fallback)
  • ☐ Нет вложенных якорей (якорь не триггерит другой якорь)
  • ☐ Description ≤ 1000 токенов, только поведенчески значимые детали
  • ☐ Price манифестируется немедленно в сцене
  • ☐ Для 12B: нет стилистических директив в Description
  • ☐ Нет действий/мыслей {{user}} в LB content
  • ☐ RepPen ≤ 1.10
