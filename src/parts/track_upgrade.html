📌 Раздел 0.3: Пути перехода между Track

Core — инструкции по расширению карточки при переходе на более высокий Track.

🚀 Track A → Track B: Расширение прототипа

Когда переходить: когда прототип работает стабильно, и вы готовы добавить глубину.

📋 Чеклист перехода

  • ⚠️ Debugging decision tree требует перехода на Track C

🔧 Шаги перехода

  1. Добавьте Price к каждому якорю — см. §R1. Price
  2. Расширьте Spine — добавьте GHOST и LIE, если их не было — см. Раздел 4.1
  3. Определите Enneagram — тип + крыло + стресс → рост — см. Раздел 4.2
  4. Дополните Examples — минимум 3 примера (нейтральный, стресс, доверие) — см. Раздел 3.2
  5. Настройте параметры — T=0.85-1.1, presence_penalty=0.0-0.15 — см. Раздел 5.1
💡 Ожидаемый результат:

Карточка ~800-1200 токенов с глубокой психологией и наблюдаемыми реакциями.

🎯 Track B → Track C: Продвинутые техники

Когда переходить: когда карточка готова к деплою, и вы хотите добавить сложные механики.

📋 Чеклист перехода

🔧 Шаги перехода

  1. Распределите GHOST по слоям — Description (определение) + LB (триггеры) + AN (состояние) — см. §R3. GHOST Placement
  2. Добавьте Lorebook — сенсорные ключи, сюжетные точки — см. Раздел 5.3
  3. Настройте Author's Notes — динамическое состояние [GHOST-активация] — см. Раздел 5.2
  4. Добавьте CoT — T2 или T3 для внутреннего монолога — см. Раздел 5.5
  5. Интегрируйте MBTI — кросс-референс с Enneagram — см. Раздел 4.4
💡 Ожидаемый результат:

Карточка ~1500-2500 токенов с полной психологической моделью, динамическим состоянием и сложными триггерами.

⚠️ Важно:

Переход на более высокий Track не означает, что предыдущий был «неправильным». Каждый Track — это осознанный выбор уровня сложности. Оставайтесь на Track A, если персонаж работает как нужно.
