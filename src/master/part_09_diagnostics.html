Критерии перехода между слоями (L3)

Когда вы готовы перейти на следующий слой?

L1 → L2

  • ☐ Карточка работает стабильно на L1
  • ☐ 2 Examples дают узнаваемый голос
  • ☐ 3-5 якорей покрывают основные ситуации
  • ☐ Нет Voice Leak
  • ☐ Хотите добавить SPINE и OCEAN

L2 → L3

  • ☐ SPINE консистентен (проверка цепочкой)
  • ☐ 3 Examples показывают разные эмоции
  • ☐ FLAW-linked якоря работают
  • ☐ Хотите добавить GHOST Layers, CoT, API блоки
  • ☐ Используете 32B+ или API модель
