Правило одного изменения

📌 Никогда не меняйте более одного параметра за раз. Иначе вы не узнаете, что именно помогло. Изменили → протестировали 5–10 сообщений → оценили → только потом следующее изменение.

Критерии перехода между слоями (L3)

Когда вы готовы перейти на следующий слой?

L1 → L2

  • ☐ Карточка работает стабильно на L1
  • ☐ 2 Examples дают узнаваемый голос
  • ☐ 3-5 якорей покрывают основные ситуации
  • ☐ Нет Voice Leak
  • ☐ Хотите добавить SPINE и OCEAN

L2 → L3

  • ☐ SPINE консистентен (проверка цепочкой)
  • ☐ 3 Examples показывают разные эмоции
  • ☐ FLAW-linked якоря работают
  • ☐ Хотите добавить GHOST Layers, CoT, API блоки
  • ☐ Используете 32B+ или API модель
  • ☐ Хотите добавить LIE (ложную установку) и GHOST (событие прошлого) к SPINE
  • ☐ Готовы добавить: CoT-якоря (2-3 максимум)
  • ☐ Готовы добавить: XML-структуру Description

Не переходите на L3, если:

  • Используете 12B модель (L3-фичи не будут работать стабильно)
  • L2-карточка ещё не протестирована
  • Не понимаете, зачем нужен GHOST/LIE
