OCEAN (Big Five) — модель из 5 личностных измерений. Используется для проверки внутренней консистентности персонажа и дополнения фреймворка SPINE (WANT (Желание)/NEED (Потребность)/FLAW (Дефект)).

📌 Золотое правило профиля: Только 1–2 экстремальных полюса OCEAN (значения <30 или >70). Каждый дополнительный экстремальный полюс снижает стабильность поведения. Нарушение = AP-15 (OCEAN Overload).

→ Подробный разбор 3 сценариев конфликта полюсов → AP-15 расширенный (Part 8, L3)

Enneagram Wings (L3)

Wing — соседний тип, который модифицирует основное поведение.

Алгоритм выбора крыла

Каждый эннеаграмм-тип имеет два смежных крыла. Выберите первичное крыло по двум критериям:

КритерийПервичное крылоВторичное крыло
ФункцияСтратегия разрешения базового страхаВариант совладания со стрессом
Пример (Тип 6)6w5: Интеллектуализация страха (анализ, подготовка)6w7: Активный поиск безопасности (связи, отвлечение)

Если персонаж реагирует на страх анализом → первичное крыло 5. Если реагирует поиском поддержки → первичное крыло 7.

OCEAN×Enneagram Matrix (L3)

Корреляция между OCEAN полюсами и Enneagram типами.

Матрица корреляции OCEAN×Enneagram. Строки: O = Открытость, C = Добросовестность, E = Экстраверсия, A = Доброжелательность, N = Нейротизм. Столбцы: Enneagram типы 1–9.

Обозначения: + = высокая корреляция, - = низкая корреляция, 0 = нейтрально.

Практическое применение

Матрица используется для валидации OCEAN→Enneagram соответствия:

  1. Определите Enneagram тип персонажа
  2. Проверьте OCEAN профиль на соответствие (+/0/- в матрице)
  3. Если профиль конфликтует — пересмотрите тип или профиль
  4. Используйте wing для уточнения профиля

Пример: Type 6w5 (Выщебленный)

Enneagram 6 (Лоялист):
- N = + (тревожность, страх без поддержки)
- E = 0 (нейтрально — может быть интроверт или экстраверт)
- A = + (лояльность = высокая доброжелательность к своим)

Wing 5 (Исследователь):
- O = + (открытость опыту, интеллектуальное любопытство)
- E = - (интроверсия, уединение)

Ожидаемый OCEAN: Высокая N, Низкая E, Высокая O, Умеренная A

Фактический OCEAN Выщебленного: O: 75, E: 20, A: 40, N: 55
✓ O: 75 = Высокая (соответствует wing 5)
✓ E: 20 = Низкая (соответствует wing 5)
✓ N: 55 = Умеренно-высокая (соответствует type 6)
~ A: 40 = Умеренная (type 6 ожидает +, но backstory объясняет снижение)

Вывод: Профиль консистентен с типом, с объяснимым отклонением по A.
