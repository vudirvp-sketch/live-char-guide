Психологический инструментарий

OCEAN (Big Five) — модель из 5 личностных измерений. Используется для проверки внутренней консистентности персонажа.

📌 Золотое правило профиля: Только 1–2 экстремальных полюса OCEAN (значения <30 или >70). Каждый дополнительный экстремальный полюс снижает стабильность поведения. Нарушение = AP-15 (OCEAN Overload).

Enneagram → SPINE

Как выводить SPINE элементы из типа Enneagram.

Enneagram элементSPINE элемент
Ключевой страх→ LIE (ложная установка, защищающая от страха) — доступна на Экспертном слое
Ключевое желание→ WANT (осознанное желание)
Направление стресса→ FLAW (поведение под стрессом)
Направление роста→ NEED (потенциал роста)

Пример: Type 6w5 (Эллиот Алдерсон)

Основное желание: Безопасность, поддержка
  ↓
WANT: Разрушить систему, которая контролирует людей

Направление стресса → Type 3 (обман, паранойя)
  ↓
FLAW: Избегание — прячется за Mr. Robot, избегает близости

Направление роста → Type 9 (спокойствие, принятие)
  ↓
NEED: Настоящая связь с людьми без масок

Основной страх: Без поддержки, без ориентира
  ↓
LIE (Ложная установка): доступна на Экспертном слое

Enneagram Wings (L3)

Wing — соседний тип, который модифицирует основное поведение.

Алгоритм выбора крыла

Каждый эннеаграмм-тип имеет два смежных крыла. Выберите первичное крыло по двум критериям:

КритерийПервичное крылоВторичное крыло
ФункцияСтратегия разрешения базового страхаВариант совладания со стрессом
Пример (Тип 6)6w5: Интеллектуализация страха (анализ, подготовка)6w7: Активный поиск безопасности (связи, отвлечение)

Если персонаж реагирует на страх анализом → первичное крыло 5. Если реагирует поиском поддержки → первичное крыло 7.

Психологический инструментарий — что дальше?

OCEAN (Big Five) — модель из 5 личностных измерений для проверки внутренней консистентности персонажа. Правило: не больше 1–2 экстремальных полюсов. Enneagram — модель из 9 типов личности для выведения СПИН из типа. Оба инструмента — интерактивные, с пентагоном и SVG-диаграммой.
