Базовые блоки карточки

Карточка персонажа — это документ, который задаёт поведение ИИ-модели при ролевой игре. Карточка состоит из четырёх блоков, каждый из которых выполняет свою функцию.

БлокФункцияВлияние на модельТокены L1Токены L2Токены L3
System PromptИнструкции, запреты, форматВысокое — задаёт рамки100-200150-250200-400
DescriptionФакты, якоря, психологияСреднее — контекст200-400400-700700-1200
ExamplesДемонстрация голосаКлючевое — паттерны речи150-250250-400400-600
GreetingПервое сообщениеПоддерживающее — тон50-10050-10050-100

📌 Важно: Description не задаёт голос. Голос задаётся ТОЛЬКО в Examples и Greeting. Description содержит только факты и поведенческие якоря.

5 базовых правил

Эти правила обязательны для всех карточек, независимо от слоя. Нарушение любого из них приводит к ухудшению качества генерации.

#ПравилоПочему важно
1OCEAN: 1-2 экстремальных полюса — значения <30 или >70Больше 2 — внутренняя противоречивость. 0 — персонаж забываемый.
2Цена обязательна — каждый FLAW/GHOST якорь имеет физическую реакциюБез цены модель не показывает уязвимость.
3Голос только в Examples — никогда в DescriptionМодель игнорирует инструкции о голосе, но копирует паттерны из Examples.
4GHOST только в Description: GHOST (событие прошлого персонажа) — только в Description, никогда в System PromptПсихология в SP ломает поведение модели.
5Формат якоря: Триггер → Действие → ЦенаКаждый элемент обязательный. Пропуск — потеря качества.

Что дальше? — Глубокий слой

СПИН (SPINE) — фреймворк из 3 базовых элементов (WANT/NEED/FLAW), который связывает мотивацию персонажа с его поведением. Якоря перестают быть случайным набором правил и получают причину. OCEAN — инструмент проверки внутренней консистентности. Всё это — на Глубоком слое.

→ СПИН, FLAW-linked якоря, OCEAN/Enneagram: SPINE Framework → Глубокий слой

Что дальше? — Экспертный слой

Если вы уже работаете с СПИН и хотите глубже — Экспертный слой добавляет LIE (ложную установку) и GHOST (событие прошлого), которые объясняют, почему персонаж ведёт себя именно так. Также: Chain of Thought, XML-структура Description, API-блоки для Claude/GPT.

→ LIE, GHOST, CoT, XML/API: Полный СПИН из 5 элементов → Экспертный слой
