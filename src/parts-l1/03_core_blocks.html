3.1 Поведенческие якоря

Якорь — это правило поведения: Триггер → Действие → Цена

Формат якоря

02
ДЕЙСТВИЕ
Наблюдаемая реакция

03
ЦЕНА
Физическая реакция
Дрожат руки

Правила якорей

  • 3-5 якорей минимум (до 7 для сложных персонажей)
  • Каждое действие = наблюдаемое, не внутреннее состояние
  • Цена обязательна для важных якорей
  • Покройте ключевые ситуации: спокойствие, стресс, радость, общение

Конкретные действия, не абстрактные прилагательные.

3.2 Примеры сообщений

Примеры — это демонстрация голоса персонажа в действии. Не описание, а показ.

Минимум 2 примера

ТипЗачемТокены
НейтральныйБазовый голос персонажа80-120
СтрессКак ведёт себя под давлением80-120
Доверие (опционально)Уязвимость, открытость80-120

Пример нейтрального сообщения

<START>
Она отложила чашку и посмотрела в окно. Дождь барабанил по стеклу.
"Опять. Третий день подряд."

Пример стрессового сообщения

<START>
Елена резко обернулась. Пальцы сжались на спинке стула.
"Ты сказал что? Повтори."

Правила написания примеров

  • Каждый блок: 80-120 токенов, одна сцена, одно состояние
  • Обязательно <START> в начале каждого блока
  • Стиль должен совпадать с Greeting
  • Не объясняй голос — покажи его

3.3 Greeting (Первое сообщение)

Greeting — первое сообщение персонажа. Задаёт место, время и тон.

Структура Greeting

  1. Сцена — где мы, что происходит (1-2 предложения)
  2. Действие — что делает персонаж
  3. Реплика — что он говорит

Пример

Библиотека погружена в вечерний полумрак. Елена сидит за столом, окружённая стопками книг. Услышав шаги, она подняла голову и сняла очки.

"О, посетитель? Редкость в это время суток."

Правила

  • Длина: 50-100 токенов
  • Тот же стиль разметки, что в Examples
  • Не начинайте с вопроса о том, что хочет пользователь

3.4 Изоляция голоса

Правило: Голос персонажа задаётся ТОЛЬКО в Examples и Greeting.

Description — только факты и якоря. Никаких стилистических директив.

Почему это важно

Модель — копировщик паттернов, не исполнитель инструкций. Если вы напишете в Description "говорит саркастично" — модель это проигнорирует. Но если в Examples персонаж говорит саркастично — модель скопирует стиль.

Иерархия влияния на голос

ИсточникВлияние
Последние сообщения в чате~85%
Примеры (Examples)Ключевое
GreetingПоддерживающее
Description0% для голоса

❌ Ошибка:

Description: "Елена говорит с сарказмом, использует слово 'чувак'"

✅ Правильно:

Description: "Елена — журналист, циничный, недоверчивый."

Example:
<START>
Она закатила глаза. "О, чувак, ещё один гений."
