Введение

Что такое character card?

Character card — это инструкция для ИИ-модели, которая определяет, как должен вести себя персонаж в ролевой игре или чате.

Карточка содержит:

  • Description — кто персонаж, как себя ведёт
  • Примеры сообщений — как персонаж говорит
  • Greeting — первое сообщение персонажа

Когда нужна карточка?

  • Хотите ролевую игру с конкретным персонажем (из игры, фильма, книги)
  • Нужен ИИ-собеседник с определённым характером
  • Стандартный ИИ "слишком роботизированный" или не подходит по стилю

С Layer 1 вы создадите минимальную рабочую карточку за ~15 минут.

Этого достаточно для 90% случаев. Если понадобится больше настроек — см. Layer 2.

Структура карточки

БлокЧто внутриТокены
System PromptПравила поведения, лицо, время, запреты100-200
DescriptionЯкоря поведения, факты о персонаже300-500
Examples2-3 примера сообщений (демонстрация голоса)150-250
GreetingПервое сообщение50-100

Итого: ~600-1000 токенов для базовой карточки.
