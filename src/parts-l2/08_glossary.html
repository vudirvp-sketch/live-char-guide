Глоссарий (Layer 2)

Все термины Layer 1 + термины Layer 2:

Базовые термины (из Layer 1)

Character Card
Инструкция для ИИ-модели, определяющая поведение персонажа.
System Prompt (SP)
Блок с правилами поведения. Задаёт ограничения, не голос.
Description
Основной блок: SPINE, якоря, факты. Только факты, не стилистика.
Anchor (Якорь)
Правило поведения: Триггер → Действие → Цена.
Price (Цена)
Мгновенная физическая реакция в той же сцене.
Examples
Демонстрация голоса персонажа. Минимум 3 для Layer 2.
Greeting
Первое сообщение персонажа. Задаёт сцену, действие, тон.
Voice Isolation
Правило: голос только в Examples/Greeting, не в Description.
Anti-Godmoding
Запрет модели описывать действия {{user}}.

SPINE Framework (Layer 2)

SPINE
Психологическое ядро персонажа: WANT, NEED, FLAW, LIE, GHOST.
WANT
Что персонаж хочет прямо сейчас. Конкретная цель или действие.
NEED
Что персонажу нужно для роста. Часто противоположно WANT.
FLAW
Изъян характера — поведение, мешающее получить NEED.
LIE
Ложная установка, во которую верит персонаж.
GHOST
Событие или паттерн из прошлого, сформировавший FLAW и LIE.

Технические термины (Layer 2)

Embodiment (Телесность)
Протокол перевода состояния в физическое действие до речи.
Format Lock
Инструкция в конце SP, фиксирующая систему разметки.
Author's Note (AN)
Короткая инструкция, вставляемая в конец контекста. Для временных изменений.
Lorebook
База знаний, вставляемая в контекст по триггерам.
Temperature
Параметр креативности. 0.7-0.85 для character cards.
Repetition Penalty
Штраф за повторы. 1.05, не выше 1.10.
Presence Penalty
Штраф за присутствие токена. Должен быть 0.0 для сохранения голоса.
Top P
Вероятностное ядро (nucleus sampling). 0.9-0.95.
Min P
Минимальная вероятность токена. 0.05.

Анти-паттерны

Token Bloat
Раздувание Description выше 800 токенов. Вызывает повторы и потерю фокуса.
Voice in Description
Стилистические директивы в Description. Конфликтуют с Examples.
Missing Price
Отсутствие или отложенность Price в якоре.
