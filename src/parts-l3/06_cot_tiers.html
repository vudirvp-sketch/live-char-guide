Chain of Thought (CoT) Tiers

Что такое CoT

Chain of Thought (CoT) — техника добавления внутреннего процесса персонажа перед реакцией.

CoT ≠ Inner Monologue

CoT — это механизм связи GHOST → реакция, не просто мысли персонажа.

CoT Tiers

УровеньФорматКогда использовать
Tier 0Без CoTLayer 1-2, простые реакции
Tier 1[Эмоция] → РеакцияLayer 2, базовая глубина
Tier 2[GHOST-связь] → РеакцияLayer 3, сложная психология
Tier 3[Полный процесс] → РеакцияLayer 3, критичные моменты

Tier 1: Эмоциональная связь

Добавляет эмоциональный контекст к реакции:

Формат:

[Эмоция: описание]
→ Наблюдаемая реакция
→ Цена

Пример:

Когда кто-то лжёт:
[Эмоция: Гнев смешивается с усталостью — опять это]
→ Сжимает челюсть, молчит
→ Цена: руки дрожат

Tier 2: GHOST-связь

Связывает реакцию с конкретным GHOST слоем:

Формат:

[Внутренний процесс: связь с GHOST]
→ Наблюдаемая реакция
→ Цена

Пример:

Когда кто-то предлагает помощь:
[Внутренний процесс: "Помощь? Как тогда. Коллега тоже 'помогал'. Все чего-то хотят."]
→ Отшучивается, говорит "я сама"
→ Цена: чувствует одиночество

Tier 3: Полный процесс

Показывает полный внутренний процесс для критичных моментов:

Формат:

[1. Триггер распознан]
[2. GHOST активирован]
[3. FLAW реагирует]
[4. LIE формирует реакцию]
→ Наблюдаемое поведение
→ Цена

Пример:

Когда спрашивают о её прошлой статье (провальной):
[1. Триггер: вопрос о провале]
[2. GHOST-2 активирован: "Источник подставил. Я не проверила."]
[3. FLAW: Защитная реакция — цинизм]
[4. LIE: "Если покажу слабость — используют"]
→ Усмехается, меняет тему: "Старая история. Что-то поинтереснее есть?"
→ Цена: голос становится жёстче, смотрит в сторону

Реализация в Description

Вариант A: Inline CoT

Якоря с CoT:

• Когда кто-то предлагает помощь:
  [GHOST-3: "Коллега тоже помогал."]
  → Отшучивается, говорит "я сама"
  → Цена: одиночество

• Когда спрашивают о провале:
  [GHOST-2: "Источник подставил."]
  → Усмехается, меняет тему
  → Цена: голос жёстче

Вариант B: CoT в отдельном блоке

<cot_rules>
Для FLAW-связанных триггеров:
1. Активировать соответствующий GHOST layer
2. Показать связь LIE → реакция
3. Цена отражает внутренний конфликт
</cot_rules>

Когда использовать CoT

СитуацияУровень
Базовые реакцииTier 0 (без CoT)
Эмоционально значимые моментыTier 1
FLAW-связанные триггерыTier 2
Критичные сюжетные моментыTier 3
Преодоление FLAW (character arc)Tier 2-3

Не перегружайте CoT!

  • Tier 2-3 только для FLAW-связанных якорей
  • Tier 3 — максимум 2-3 якоря
  • Базовые реакции без CoT

Пример: Полный якорь с CoT Tier 2

Когда кто-то говорит "Я просто хочу помочь":

[Внутренний процесс]:
"GHOST-3 активирован. Коллега тоже 'просто хотел помочь' — и украл материал.
GHOST-2: Источник тоже 'просто хотел помочь' — и подставил.
Все лгут. Все чего-то хотят."

→ Елена резко выпрямляется. Взгляд становится холоднее.
"У каждого есть мотив. Какой твой?"

Цена: Напряжение в челюсти. Она знает, что отталкивает человека, который может быть искренен. Но не может позволить себе риск.
