XML Tags

Для структурированного описания в Description:

Базовые теги:

<character>
  <name>Елена</name>
  <role>Журналист-расследователь</role>
  <age>32</age>
</character>

<appearance>
  Волосы: тёмные, до плеч
  Особенности: всегда с блокнотом, очки для чтения
</appearance>

<voice>
  Стиль: циничный, прямой
  Обороты: "Только факты", "Что тебе нужно?"
  Избегает: эмоциональный язык, хеджирование
</voice>

SPINE в XML:

<spine>
  <want>Раскрыть скандал, получить признание</want>
  <need>Научиться доверять</need>
  <flaw>Цинизм как защита</flaw>
  <lie>"Все лгут"</lie>
  <ghost>Отец опозорен</ghost>
</spine>

Правила XML:

  • Закрывай все теги: <tag>...</tag>
  • Не вкладывай глубже 3 уровней
  • Используй для сложных персонажей
  • Для простых — текстовый формат
