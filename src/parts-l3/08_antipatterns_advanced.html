Advanced Antipatterns

Layer 3 добавляет анти-паттерны для сложных конструктов.

Сложные анти-паттерны

❌ AP-8: GHOST без якорей

ПРОБЛЕМА: GHOST описан, но не имеет связанных якорей → "мёртвый" контент.

РЕШЕНИЕ: Каждый GHOST layer = минимум 1 якорь.

ПРИМЕР:

❌ GHOST описан, но в якорях нет триггеров, связанных с ним

✅ GHOST: "Коллега сдал"
✅ Якорь: "Когда новый коллега предлагает помощь → подозревает мотив"

❌ AP-9: SPINE разорван

ПРОБЛЕМА: Элементы SPINE не связаны логически → персонаж несогласован.

РЕШЕНИЕ: Проверь цепочку GHOST → LIE → FLAW → NEED → WANT.

ПРОВЕРКА:

❌ GHOST: "Отец ушёл"
   FLAW: "Грызёт ногти"
   → Нет связи!

✅ GHOST: "Отец ушёл, потому что мама контролировала"
   LIE: "Контроль = безопасность"
   FLAW: "Перехватывает контроль, перебивает"
   → Логичная цепочка

❌ AP-10: CoT Overload

ПРОБЛЕМА: CoT добавлен к каждому якорю → перегрузка, модель путается.

РЕШЕНИЕ: CoT Tier 2-3 только для FLAW-связанных якорей (2-3 максимум).

❌ AP-11: Multi-char Voice Bleed

ПРОБЛЕМА: В multi-char карточках голоса персонажей смешиваются.

РЕШЕНИЕ: Чёткие маркеры персонажей + разные стили разметки.

ПРИМЕР:

❌ Оба персонажа в одном стиле:
<START>
"Привет." — сказала Елена.
"Привет." — ответил Макс.

✅ Разные стили:
<START> [Елена]
Она закатила глаза. "О, ещё один."

<START> [Макс]
Он усмехнулся, прислонившись к стене.
"Спокойно. Я не враг."

❌ AP-12: XML Malformed

ПРОБЛЕМА: Незакрытые XML теги → модель не парсит структуру.

РЕШЕНИЕ: Всегда закрывай теги: <tag>...</tag>

❌ AP-13: Lorebook Conflict

ПРОБЛЕМА: Lorebook содержит информацию, конфликтующую с Description.

РЕШЕНИЕ: Lorebook = дополнение, не противоречие.

❌ AP-14: Context Window Violation

ПРОБЛЕМА: Description + Lorebook + Examples превышают контекст.

РЕШЕНИЕ: Для 4K контекста: Description ≤ 400, Examples ≤ 200, Lorebook минимальный.
